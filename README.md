## Installation
* This project using `pnpm` as package manager.
# If you don't have pnpm installed, 
    run: npm install -g pnpm
    
```bash
    npm install -g pnpm
    git clone https://github.com/billymaulana/vue3-checkoutApps.git
    cd vue3-checkoutApps
    pnpm install
    pnpm build
    pnpm dev
```
